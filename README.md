# Customer Segmentation 
**Algonive Data Analytics Internship Project**

## Overview
A customer segmentation system that categorizes 200 customers into 5 behavioural segments based on purchasing behaviour, demographics, and preferences using clustering techniques.
Datset-Kaggle (https://www.kaggle.com/datasets/hosseinbadrnezhad/mall-customer-segmentation-dataset)

## Segments Identified
- 🔵 Budget Customers (40.5%)
- 🟢 Premium Customers (19.5%)
- 🟠 Impulsive Customers (17.5%)
- 🩷 Average Customers (11.5%)
- 🟣 Conservative Customers (11.0%)

## Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Clustering & Segmentation
- Interactive Dashboard (HTML)
- Business Recommendations Report

## Files
| File | Description |
|------|-------------|
| `Customer_Segmentation.csv` | raw dataset |
| `customer_segmentation_dashboard.html` | Interactive dashboard — open in browser |
| `customer_data.csv` | Cleaned & processed dataset |

## How to Run
1. Download `customer_segmentation_dashboard.html`
2. Open in any browser (Chrome/Edge recommended)
3. Use filters to explore segments interactively
4. Click **Export Report** to generate the PDF report

## Tools Used
- Python (Pandas, Scikit-learn, Matplotlib)
- HTML / CSS / JavaScript
- Chart.js
